## Notes
- **Multi-page tables**: Tables 1–3 each span several pages; the screenshot is one representative
  page (with the caption) and the markdown holds the full transcribed row set.
- **Aggregate numbers** for the claims (counts/percentages by phase, purpose, CADRO, sponsor,
  biomarker, participants, geography, repurposing, trajectory) are reported in the paper's Results
  text (§3.1–§3.12) and summarized in Figures 2–5; the per-agent detail lives in Tables 1–3.
- **Flagged source inconsistency**: Figure 5B's plotted Tau line ends near ≈9%, whereas §3.5 text
  states tau rose "to approximately 20%". See figures/figure5.md. C10 quotes the text verbatim.
- **Figure 3 vs §3.1 counts**: Figure 3 gives per-phase agent counts; agents assessed in >1 phase
  contribute to multiple segments, so its row totals run ~1 higher than the once-per-pipeline totals
  in §3.1 (e.g., transmitter receptors 39 vs 38). See figures/figure3.md.
