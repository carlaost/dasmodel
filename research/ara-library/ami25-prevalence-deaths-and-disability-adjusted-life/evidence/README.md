## Evidence-completeness accounting

**Numbered objects present in the provided PDF (main article):** Table 1; Figure 1; Figure 2;
Figure 3; Figure 4. All five are filed above with BOTH a markdown transcription/description and a
`.png` screenshot. No numbered main-text object was skipped.

**Supplementary objects referenced but NOT in the provided input:** The text cites Tables S1, S2, S3
and Figures S1, S2, S3 (per-country prevalence/death/DALY ASRs and their percentage-change maps).
These live in the online Supplementary Information (https://doi.org/10.1038/s41598-025-89899-w) and
were **not part of the provided input** (only `paper.pdf` = the main article was supplied). They are
therefore not filed as evidence objects. The per-country 2021 ASRs that S1–S3 would contain are
already fully captured at one-decimal precision in Table 1 (and at two-decimal precision for the
extreme countries in the Table 1 running-text note). No fabrication of S-material was performed.
