## Visual description
- **Components** (top to bottom, grouped into four PRISMA stages via labeled side-bars):
  - **Identification** (yellow side-bar): "Records identified through database searches (n = 1748)" box; "Records identified through reference list (n = 4)" box, feeding into it.
  - **Screening** (green side-bar): "Records after duplicates removed (n = 1202)" box → "Titles & Abstracts screened (n = 1202)" box → "Titles & Abstracts excluded (n = 1129)" box (side branch).
  - **Eligibility** (orange side-bar): "Full-text articles assessed for eligibility (n = 73)" box → "Full-text articles excluded (n = 44)" box (side branch), itemising: "Did not analyse barriers (n=27)"; "Study design/type (n=12)"; "Not set in a European nation (n=4)"; "Different population (n=1)".
  - **Included** (blue side-bar): "Studies included in final analysis (n = 29)" box.
- **Connections**: Vertical arrows show the main funnel (1748+4 → 1202 → 1202 → 73 → 29); horizontal arrows branch off to the two exclusion boxes ("Titles & Abstracts excluded" and "Full-text articles excluded") at the screening and eligibility stages respectively.
- **Annotations**: Colored vertical bars label each of the four PRISMA stages (Identification = yellow, Screening = green, Eligibility = orange, Included = blue); the "Full-text articles excluded" box uses a bulleted sub-list to break down the n=44 exclusions by reason.
- **What it conveys**: The complete, staged record-attrition pipeline from initial identification (1752 total: 1748 database + 4 reference-list) down to the 29 studies included in the final narrative synthesis, with exact counts and reasons at each exclusion point.
