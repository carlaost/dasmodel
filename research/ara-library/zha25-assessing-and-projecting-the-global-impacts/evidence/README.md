Figure 3; Figure 4. All five are filed above with BOTH a markdown transcription/description and a
`.png` screenshot. No numbered main-text object was skipped. The paper contains exactly one

**Supplementary objects referenced but NOT in the provided input:** The Results text cites
Supplementary Table 1 (1990/2030 global point ASRs by measure); Supplementary Tables 2, 5
(2030 region-level ASRs); Supplementary Table 4 (region-level EAPCs); Supplementary Tables 6, 8
(country-level EAPCs); Supplementary Tables 7, 9 (country-level 2030 ASRs); Supplementary Figure 1
(country-level EAPC/ASR maps); Supplementary Figures 2, 3 (SDI-threshold group comparisons at 0.8
and 0.7); Supplementary Figure 4 (5-category SDI comparison); Supplementary Figure 5 (per-continent
SDI-correlation scatter plots). These live in the online Supplementary Information
(https://www.frontiersin.org/articles/10.3389/fpubh.2024.1453489/full#supplementary-material) and
were **not part of the provided input** (only `paper.pdf` = the main article was supplied). They
are therefore not filed as evidence objects. The specific region- and country-level extreme values
these supplementary objects would tabulate in full are transcribed, where the running text states
them, into claims C03–C08 and cited to the Results section text (not to a table/figure file) —
this is noted explicitly in each such claim's `Evidence basis`. No fabrication of S-material was
performed; exact statistics not stated in the main text (e.g., Mann-Whitney U values/p-values for
the SDI-threshold comparisons in C07) are marked "Not specified in paper."
