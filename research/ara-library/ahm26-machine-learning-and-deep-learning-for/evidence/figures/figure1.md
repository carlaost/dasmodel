## Visual description
- **Components**: Three dataset boxes on the left — **ADNI**, **OASIS / OASIS-3**, **AIBL** — each
  linked to a list of associated modalities on the right. A separate panel lists "Typical Research Uses."
  - ADNI → MRI, PET, Clinical / Cognitive Data, Genetic / Biomarker Data
  - OASIS / OASIS-3 → MRI, PET, Clinical / Cognitive Data
  - AIBL → MRI, PET, Clinical / Cognitive Data, Genetic / Biomarker Data
  - Typical Research Uses → Alzheimer's disease classification; Mild cognitive impairment detection;
    Progression prediction; Multimodal learning; External validation
- **Connections**: Arrows from each dataset box to its modality list.
- **Annotations**: MRI and PET rows are color-shaded; the title reads "Summarized public datasets and
  their modalities."
- **What it conveys**: The field's dominant public benchmark cohorts and the multimodal (imaging +
  non-imaging) data they provide; emphasizes that progress is driven by data availability/integration,
  not only architecture. Supports O2, O5, and the data-sources list in src/environment.md.
