## Abstract

Alzheimer's disease (AD) is highly heterogeneous, with characteristic regional changes in patterns of cortical volume. This study aims to predict future longitudinal brain volume changes in older individuals with various stages of clinical diagnoses, from healthy to dementia, using predictions from baseline MRI. We test two deep learning (DL) approaches predicting longitudinal atrophy: (i) a unimodal model using baseline MRI scans from a single time point as predictor and log-Jacobian atrophy map as target, and (ii) a multimodal model including imaging and non-imaging data. We then developed a normative model using mixed linear regression, based on the longitudinal brain changes predicted by the DL model, to detect deviations in the clinical patient population. This preliminary result using the Alzheimer's Disease Neuroimaging Initiative (ADNI) dataset demonstrated the efficacy of the proposed framework to detect abnormal brain regions with high atrophy rate related to dementia.
